# Asistencia-Estudiantil-Por-Carnet-
Aplicación desarollada en Python para el registro automático de asistencia estudiantil a las diferentes clases mediante el carnet de la institución.

Objetivo:
Poder optimizar el control de la asistencia a los estudiantes a las clases mediante el carnet ayudando a que todo sea más rapido para aprovechar el tiempo y aumentar la responsabilidad de los estudiantes. 

Funcionalidades:
-Registro de asistencia a las clases solo con escanear el carnet
- Guardado automático del registro de asistencia para la suma de la nota
- Uso en entorno educativo real 

Tecnologías:
-Python
-Lectura de carnet por escaner en tiempo real

Nota de Seguridad:
Este repositorio contiene una versión demostrativa del sistema.
Algunas funciones fueron omitidas o no sirven al 100% por motivos de seguridad
